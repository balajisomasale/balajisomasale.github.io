Explorer Bot | War Field Spying Robot

Nov 2016 – Dec 2016


Project description

  • This Project was mainly built with 'GSM module' & ‘XBEE’ modules designed for Communication (Message Alert) & motion Navigation as Primary Objective & Night-vision Wireless camera with Ultrasonic sensors as secondary objectives.
  • This Robot is mainly used for the purpose of Exploring fields where humans are restricted to go. If both the XBEE and the GSM Module fails the robot is designed to become autonomous.
  • XBEE is a wireless transmitting device (consists of both Transmitter and receiver).It has a built in camera to observe everything around and also some attacking features which restricts from damages.
  • Upgraded Version of Bluetooth Module robots.

  • Got BEST RESEARCH AWARD for EXPLORER BOT (War Field Spying Robot) in Robotic and Automation for Humanitarian Applications (RAHA2016) International Conference on 18th December 2016 to 20 December 2016
